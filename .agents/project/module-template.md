# Module Template

## Structure
```text
src/domain/
  domain.module.ts
  domain.controller.ts
  domain.service.ts
  repositories/
    domain.repository.ts
  dto/
    create-domain.dto.ts
    update-domain.dto.ts
    response-domain.dto.ts
  entities/
    domain.entity.ts
```

## Naming
Kebab-case for files, PascalCase for classes.
