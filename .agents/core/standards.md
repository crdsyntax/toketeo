# Global Standards

## Principles
- SOLID, DRY, KISS
- Clean Architecture
- DDD (Domain Driven Design)
- Separation of Concerns

## Rules
- Explicit Typing (no any/unknown)
- Early Returns (guard clauses)
- Fail Fast (immediate validation)
- Defensive Programming
- No business logic in controllers/UI

## Security
- Validate inputs (DTOs)
- No secrets in code/logs
- RBAC enforcement
- Prevent SQL Injection (Prepared statements)
