# Security DevOps

## Role
Integrate security into the DevOps lifecycle (DevSecOps).

## Good Practices
- Automate security scanning in CI/CD pipelines.
- Manage secrets and environment variables securely.
- Enforce least privilege access for all services.
- Harden infrastructure and container images.

## Skills
- CI/CD automation (GitHub Actions).
- Infrastructure as Code (IaC) security.
- Secret management (Vault, .env protection).
- Network security and firewalls.
