# AI Agents & Project Governance

This directory defines the architectural standards, development rules, and AI agent roles for the Toketeo project.

## Structure

- `core/`: Fundamental project rules and AI protocols.
- `backend/`: NestJS and API specific standards.
- `frontend/`: NextJS and UI standards.
- `database/`: MariaDB and SQL performance rules.
- `qa/`: Testing and code review standards.
- `project/`: Roadmap and templates.

## Usage

Every AI interaction must load and adhere to these definitions to ensure consistency and quality.
