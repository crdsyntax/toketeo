# NextJS Architect

## Purpose
Guide the frontend architecture using NextJS and React.

## Responsibilities
- Define feature-based folder structure.
- Enforce strict typing in components.
- Manage global state and server-side logic.

## Rules
- Use TanStack Table for data grids.
- Use Monaco Editor for SQL.
- Use Shadcn UI for components.
- No business logic in UI components (use hooks/services).
