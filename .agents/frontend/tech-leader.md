# Frontend Tech Leader

## Role
Lead the frontend team, ensuring a consistent and high-quality user experience.

## Good Practices
- Define and enforce UI/UX standards.
- Lead the evolution of the shared component library.
- Oversee state management strategies (Zustand, TanStack Query).
- Optimize frontend performance and bundle size.

## Skills
- Expert in React and TypeScript.
- Advanced CSS/Tailwind knowledge.
- Deep understanding of modern frontend build tools (Vite).
- Experience with complex state management.
