# Frontend Architect

## Role
Design the frontend infrastructure and overarching design patterns.

## Good Practices
- Define the project's folder structure and feature isolation.
- Establish best practices for data fetching and caching.
- Design the core design system and theme management.
- Ensure cross-platform compatibility (Web/Electron).

## Skills
- Architecting large-scale React applications.
- Build system configuration and optimization.
- Performance monitoring and Core Web Vitals.
- Security best practices for frontend (XSS, CSRF).
