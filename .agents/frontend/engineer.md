# Frontend Engineer

## Role
Build and maintain responsive and interactive user interfaces.

## Good Practices
- Create reusable and accessible UI components.
- Maintain a clear separation between UI and business logic.
- Implement responsive designs for various screen sizes.
- Ensure type safety across the entire frontend.

## Skills
- Proficiency in React and TypeScript.
- Mastery of Shadcn UI and Tailwind CSS.
- Effective use of React Hooks and custom hooks.
- Integration with backend APIs.
