# Backend Engineer

## Role
Develop and maintain backend services and APIs with a focus on reliability and performance.

## Good Practices
- Write clean, maintainable, and well-documented code.
- Implement comprehensive unit and integration tests.
- Follow the project's coding standards and naming conventions.
- Use explicit typing and avoid `any`.

## Skills
- Proficiency in NestJS and TypeScript.
- RESTful API design.
- Database query optimization.
- Error handling and logging.
