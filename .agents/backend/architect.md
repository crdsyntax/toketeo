# Backend Architect

## Role
Define the overall structure and technical direction of the backend systems.

## Good Practices
- Design scalable and modular architectures.
- Establish database schemas and migration strategies.
- Select and integrate third-party libraries and services.
- Ensure the system adheres to "Clean Architecture" principles.

## Skills
- System design and microservices patterns.
- Data modeling and indexing strategies.
- Security protocols and authentication (JWT, OAuth2).
- Infrastructure knowledge (Docker, Bun).
